## Ini harus di-install tergantung pada OS yang digunakan:
✿ `onnx`<br>
✿ `onnxruntime`<br>
✿ `opencv-python`<br>
✿ `pillow`<br>
✿ `numpy`<br>
✿ `python-multipart`<br>
✿ `fastapi`<br>
✿ `uvicorn`<br>
<br>
misalnya kalau pake nyarch linux bisa pake command:
```bash
$ yay -S python-onnx
$ yay -S python-onnxruntime
$ yay -S python-opencv
$ yay -S python-pillow
$ yay -S python-numpy
$ yay -S python-python-multipart
$ yay -S python-fastapi
$ yay -S uvicorn
```
<br>

untuk selain OS distribusi arch cukup di-install melalui pip:
```bash
$ pip install onnx onnxruntime opencv-python pillow numpy python-multipart fastapi
$ sudo apt install uvicorn
```
